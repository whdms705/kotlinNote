# null 안전과 예외

## 코틀린의 null 처리 개요

* 코틀린에는 null 값이 지정될 수 있는 요소가 있는가 하면 그렇지 않은 요소도 있다.

* String은 null 불가능 타입이며 컴파일러가 null 값의 지정을 막는다.

* 코틀린은 null 값을 가질 수 있다고 특별히 지정되지 않으면 변수가 null을 가질수 없다.
따라서 , null 값으로 생기는 문제를 런타임이 아닌 컴파일 시점에 방지할 수 있다.
